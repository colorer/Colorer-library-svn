Настройки схем

главная

Диалог настроек схем вызывается из окна настроек FarColorer кнопкой Настройки схем.


В данном окне настраиваются пользовательские значения параметров для каждой схемы индивидуально. Настройки сохраняются в реестре с остальными настройками плагина. Типы параметров и их значения, а так же порядок наследования приведены в разделе Параметры схем.

Большинство параметров принимает фиксированные значения. Все эти значения доступны для выбора в поле Значение параметра. Для остальных параметров, например backparse, значение нужно вводить вручную. Так же можно указать, чтобы для параметра устанавливалось значение по умолчанию, выбрав в поле <default-знач>.

Если для параметра пользователем не установлено значение, то в поле ввода значения отображается текст <default-знач>. Данный текст показывает, что для параметра по умолчанию будет выставлено значение знач. Данное значение получено путем наследования.

Сохранение всех изменений происходит после нажатия кнопки OK. Кнопка Отмена не сохраняет никаких изменений.
