FAR File Manager plugin

Installation

To install colorer plugin for FAR Manager, you must copy all the content of library archive into the Far\Plugins\Colorer folder and restart FAR Manager. Be sure to remove or disable all previous versions of colorer, installed in FAR.

Features

Plugin for FAR Manager implements most Colorer-take5 features. These include:

  • Editor syntax highlighting capabilities.
  • Paired structures highlighting, search and selection.
  • Outline structure creating and flexible management.
  • Syntax caching, on modify reparsing.

You can list and check all types, available in current HRC set. Plugins supports selection of one of the installed Coloring styles.

For more information see FAR Manager help files, associated with plugin.
