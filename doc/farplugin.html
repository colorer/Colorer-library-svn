FAR File Manager plugin

Installation

Get FARColorer

To install colorer plugin for FAR Manager, you have to copy all the content of the plugin pack into the Far\Plugins\Colorer folder and restart FAR Manager. Be sure to remove or disable all previous versions of colorer, installed in FAR.

Features

Colorer Plugin for FAR Manager implements all the most important Colorer-take5 features. These include:

  • Editor syntax highlighting capabilities.
  • Paired structures highlighting, search and selection.
  • Outline structure creation and flexible management.
  • Syntax caching, on modify reparsing.

You can list and check all types, available in current HRC set. Plugins supports selection of one of the installed Coloring styles.

For more information see FAR Manager help files, associated with plugin.
