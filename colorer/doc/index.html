Welcome to the Colorer Library project

Introduction

Colorer take5 is a syntax highlighting and text parsing library, that provides services of text parsing in host editor systems in real-time and transforming results into colored text. Result information also allows to search and build outlined lists of special text tokens (function lists, syntax errors) and to search and indent programming language constructions (brackets, paired tags). Colorer uses pure C++ (all internal syntax structures are stored in XML), it is fully portable and works on either win32/unix/mac platforms. Top level Java language API is also available.

Highlighting

Colorer supports highlighting and editing capabilities of numerous programming languages, scripts and markups. These sets are stored in the internal language (XML-based HRC files), describing target's language syntax. HRC database is open and ready to modify and growth.
