Установка FarColorer

главная
  1. Скачиваем последнюю версию FarColorer с оффициального сайта.
  2. Распаковываем содержимое архива в подпапку Plugins папки с утановленным Far2.
  3. Запустить Far ( либо перезапустить, если установка шла в нем ). Теперь, если открыть файл, например, файл LICENSE, то вы увидете раскрашенный текст
